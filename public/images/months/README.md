# Monthly Images Directory

## Adding Images for Each Month

To add images for each month, place image files in this directory with the following naming convention:

- `january.jpg` (or `.png`, `.webp`, etc.)
- `february.jpg`
- `march.jpg`
- `april.jpg`
- `may.jpg`
- `june.jpg`
- `july.jpg`
- `august.jpg`
- `september.jpg`
- `october.jpg`
- `november.jpg`
- `december.jpg`

## Supported Formats

- JPG/JPEG
- PNG
- WebP
- GIF

## Image Recommendations

- Recommended size: 1200x800 pixels or similar aspect ratio
- File size: Keep under 500KB for optimal performance
- Format: JPG for photos, PNG for graphics with transparency

The app will automatically display the image for each month when you navigate to that month in the calendar.
